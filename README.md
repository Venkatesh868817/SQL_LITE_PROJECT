# SQLite Practice Project

## 📌 Project Overview

This project demonstrates the use of **SQLite and SQL queries** to explore and analyze data from a sample database. The main goal of this project is to practice fundamental SQL operations and understand how relational databases work.

## 🗂 Database Tables Used

The project includes the following tables:

* Customers
* Orders
* Employees
* Suppliers
* Products
* Regions
* CustomerCustomerDemo

These tables simulate a small business database and help in learning database relationships and data handling.

## ⚙️ Features

* Data retrieval from multiple tables
* Sorting and filtering records
* Working with relational data
* Understanding database structure using SQLite

## 🛠 Tools & Technologies

* SQLite
* SQL
* DB Browser for SQLite
